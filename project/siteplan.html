Site Plan

DYNAMIC WEATHER

Proposed name: Dynamic Weather.

Reason for selection: It offers a practical and dynamic user experience. Weather data is something that users often need in real time.

Site Purpose

The project is a weather website that dynamically displays weather information based on the user's current area or a city they enter.

Scenarios

  • What is the current weather in my city?
  • Where can I find weather forecasts for the next week?

Selected colors:
